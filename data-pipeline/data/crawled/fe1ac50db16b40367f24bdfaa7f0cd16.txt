## Welcome

### What is Vector DB Comparison?

Vector DB Comparison is a free and open source tool from Superlinked to compare vector databases. Each of the features outlined has been verified to varying degrees by the community.

[Superlinked](https://github.com/superlinked/superlinked) is a compute framework for turning multimodal data structures into vector embeddings

4. Drag columns next to vendor column to pin them

![](/vector-db-comparison/_next/image?url=%2Fvector-db-comparison%2F_next%2Fstatic%2Fmedia%2Fhow-to-use-5.47fa7798.gif&w=3840&q=75)
