#  [ ](#dataset-card-for-capybara-preferences) Dataset Card for Capybara-Preferences 

This dataset has been created with [distilabel](https://distilabel.argilla.io/).

![](https://cdn-uploads.huggingface.co/production/uploads/60420dccc15e823a685f2b03/Vmr0FtTvnny6Snm-UDM_n.png)

##  [ ](#dataset-summary) Dataset Summary 

This dataset is built on top of [`LDJnr/Capybara`](https://huggingface.co/datasets/LDJnr/Capybara), in order to generate a preference dataset out of an instruction-following dataset. This is done by keeping the conversations in the column `conversation` but splitting the last assistant turn from it, so that the conversation contains all the turns up until the last user's turn, so that it can be reused for generating alternative responses with other models i.e. the last assistant turn. The existing completion for each conversation is kept, so as to also include that within the UltraFeedback critique, so that we end up comparing the existing generation and the alternative generations.

The dataset contains multi-turn conversations, and the follow up turns (without taking into consideration the first turn) are generated synthetically via `gpt-4-1106-preview` with the `Amplify-Instruct` method developed by LDJ et al. (still to be released). Then we generate alternative completions for those conversations with the following 7B LLMs: `argilla/notus-7b-v1`, `mlabonne/NeuralBeagle14-7B`, and `teknium/OpenHermes-2.5-Mistral-7B`; and then all those are provided to UltraFeedback for the critique, so that `gpt-4-1106-preview` is used as a judge, providing both ratings and rationales for each response i.e. last assistant turn, out of the four existing ones.

##  [ ](#reproduce) Reproduce 

The dataset has been synthetically generated with few to no human intervention in the process, besides some pre and post processing, using [`distilabel`](https://github.com/argilla-io/distilabel). More about `distilabel` at <https://distilabel.argilla.io/latest/>.

First of all, in order to reproduce the following pipeline, you should install `distilabel` as follows:

```
`pip install "distilabel[openai,vllm]" --quiet `
```

Additionally, for speed ups when using vLLM, you can also install Flash Attention as follows:

```
`pip install flash-attn --no-build-isolation `
```

And set the `OPENAI_API_KEY` environment variable value, required to run UltraFeedback; and the `HF_TOKEN` required to finally push the generated `distilabel.Distiset` into the Hugging Face Hub.

The complete `distilabel.Pipeline` is available in the [`example.py`](https://huggingface.co/datasets/argilla/Capybara-Preferences/blob/main/example.py) file within this repository.

###  [ ](#code) Code 

We will break down the process in three stages:

**1. Define Custom `Step`**

```
`from distilabel.steps import StepInput, step from distilabel.steps.typing import StepOutput # Custom Step to remove the last assistant response from `messages`, while preserving it under `response` @step(inputs=["messages"], outputs=["messages", "response"]) def FormatMessages(*inputs: StepInput) -> StepOutput: for input in inputs: for item in input: messages = [] for i in item["messages"]: messages.extend( [ {"role": "user", "content": i["input"]}, {"role": "assistant", "content": i["output"]}, ] ) item.update({"messages": messages[:-1], "response": messages[-1]["content"]}) yield input # Custom Step to preserve the existing generation from LDJnr/Capybara @step(inputs=["response"], outputs=["generation", "model_name"]) def PreserveGeneration(*inputs: StepInput) -> StepOutput: for input in inputs: for item in input: item.update({"generation": item["response"], "model_name": "gpt-4-1106-preview"}) yield input # Custom Step to format the `messages` into a single string as the `instruction` before calling OpenAI @step(inputs=["messages"], outputs=["instruction"]) def PrepareInstruction(*inputs: StepInput) -> StepOutput: for input in inputs: for item in input: if item["messages"][-1]["role"] == "assistant": item["messages"] = item["messages"][:-1] item["instruction"] = "".join([f"<|{i['role']}|>\n{i['content']}</s>\n" for i in item["messages"]]) yield input `
```

**2. Define `Pipeline`**

The `Pipeline` is defined as follows:

```
`load_hub_dataset \ > format_messages \ > [text_generation_with_openhermes, text_generation_with_notus, text_generation_with_neuralbeagle, preserve_generation] \ > combine_columns \ > prepare_instruction \ > ultrafeedback \ > keep_columns `
```

And the code looks like:

```
`import os from distilabel.llms import OpenAILLM, vLLM from distilabel.pipeline import Pipeline from distilabel.steps import ( CombineColumns, KeepColumns, LoadHubDataset, ) from distilabel.steps.tasks import TextGeneration, UltraFeedback with Pipeline(name="capybara-preferences") as pipeline: # First we load the `Capybara` dataset from the Hub, mapping the column conversation into messages load_hub_dataset = LoadHubDataset( name="load_dataset", output_mappings={"conversation": "messages"}, ) # Then we format the messages so as to split the messages from the last assistant response format_messages = FormatMessages(name="format_messages") load_hub_dataset.connect(format_messages) # Then we run the generation using `teknium/OpenHermes-2.5-Mistral-7B` via vLLM text_generation_with_openhermes = TextGeneration( name="text_generation_with_openhermes", llm=vLLM( model="teknium/OpenHermes-2.5-Mistral-7B", model_kwargs={"trust_remote_code": True, "dtype": "auto"}, ), input_batch_size=10, input_mappings={"instruction": "messages"}, output_mappings={"model_name": "generation_model"}, ) format_messages.connect(text_generation_with_openhermes) # Then we run the generation using `argilla/notus-7b-v1` via vLLM text_generation_with_notus = TextGeneration( name="text_generation_with_notus", llm=vLLM( model="argilla/notus-7b-v1", model_kwargs={"dtype": "auto"}, ), input_batch_size=10, input_mappings={"instruction": "messages"}, output_mappings={"model_name": "generation_model"}, ) format_messages.connect(text_generation_with_notus) # Then we run the generation using `mlabonne/NeuralBeagle14-7B` via vLLM text_generation_with_neuralbeagle = TextGeneration( name="text_generation_with_neuralbeagle", llm=vLLM( model="mlabonne/NeuralBeagle14-7B", model_kwargs={"dtype": "auto"}, ), input_batch_size=10, input_mappings={"instruction": "messages"}, output_mappings={"model_name": "generation_model"}, ) format_messages.connect(text_generation_with_neuralbeagle) # Then we run the custom step to preserve the existing generation from Capybara preserve_generation = PreserveGeneration( name="preserve_generation", input_batch_size=10, output_mappings={"model_name": "generation_model"}, ) format_messages.connect(preserve_generation) # Then we merge the values for `generation` together into a single list combine_columns = CombineColumns( name="combine_columns", columns=["generation", "generation_model"], output_columns=["generations", "generation_models"], ) text_generation_with_openhermes.connect(combine_columns) text_generation_with_notus.connect(combine_columns) text_generation_with_neuralbeagle.connect(combine_columns) preserve_generation.connect(combine_columns) # Before calling OpenAI with UltraFeedback, we transform the messages into a string i.e. instruction prepare_instruction = PrepareInstruction(name="prepare_instruction") combine_columns.connect(prepare_instruction) # Then we run the UltraFeedback task ultrafeedback_openai = UltraFeedback( name="ultrafeedback_openai", llm=OpenAILLM( model="gpt-4-1106-preview", api_key=os.getenv("OPENAI_API_KEY"), ), aspect="overall-rating", output_mappings={"model_name": "ultrafeedback_model"}, ) prepare_instruction.connect(ultrafeedback_openai) # Finally, we filter out / remove the columns that we don't want by specifying the ones we want keep_columns = KeepColumns( name="keep_columns", columns=[ "source", "messages", "generations", "generation_models", "ratings", "rationales", "ultrafeedback_model", ], ) ultrafeedback_openai.connect(keep_columns) `
```

**3. Running `Pipeline`******

```
`distiset = pipeline.run( parameters={ "load_dataset": { "repo_id": "LDJnr/Capybara", "split": "train", }, "text_generation_with_openhermes": { "llm": { "generation_kwargs": { "max_new_tokens": 1024, "temperature": 1, }, }, }, "text_generation_with_notus": { "llm": { "generation_kwargs": { "max_new_tokens": 1024, "temperature": 1, }, }, }, "text_generation_with_neuralbeagle": { "llm": { "generation_kwargs": { "max_new_tokens": 1024, "temperature": 1, }, }, }, "ultrafeedback_openai": { "llm": { "generation_kwargs": { "max_new_tokens": 1024, "temperature": 1, }, }, }, } ) if distiset is not None: distiset.push_to_hub( "Capybara-Preferences", token=os.getenv("HF_TOKEN"), ) `
```
